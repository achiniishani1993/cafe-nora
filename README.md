# cafe-nora
Create cafe website using html and css 

Cafe Nora is a modern, elegant multi-page café website built using HTML5 and CSS3. The project showcases a complete front-end layout including a homepage, gallery, and menu page with structured layouts, animations, and interactive elements.

# Project Overview

This project represents a fully designed café website featuring:

* A welcoming homepage with slideshow and booking system

* A visually rich gallery page

* A structured and styled menu page

* Contact section with embedded location map

* Footer with newsletter subscription

* CSS animations and hover effects

* Grid and Flexbox layout systems

The design focuses on elegance, readability, and user engagement.


# Animations & Effects

* Slideshow animation using @keyframes

* Hover zoom effect on gallery images

* Animated accent line on menu items

* Popup form display using :target

* Map hover scaling effect

* Button hover transitions


# Technologies Used

* HTML5

* CSS3

* CSS Grid

* Flexbox

* Google Fonts

* Embedded Google Maps

* Font Awesome icons


# Forms Included

* Table reservation form

* Contact form

* Newsletter subscription form

All forms use built-in HTML validation attributes.
